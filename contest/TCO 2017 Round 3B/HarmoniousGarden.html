TCO 2017 Round 3B 1 250 HarmoniousGarden

Problem Statement

Fox Ciel wants to build a garden. In this task, a garden consists of some pavilions and some paths. Each path is bidirectional and connects two different pavilions. Each pair of pavilions can be directly connected by at most one path.

A simple cycle is a sequence (x1, x2, ..., xm) of three or more mutually distinct pavilions such that the garden contains all of the following paths: (x1, x2), (x2, x3), ..., (xm-1, xm), and also (xm, x1). The length of a simple cycle is the number of pavilions it contains.

Two cycles are considered the same if we can rotate and/or flip one of them to get the other. For example, (x1, x2, x3, x4), (x2, x3, x4, x1), and (x4, x3, x2, x1) are all the same cycle, but (x1, x3, x2, x4) is another cycle that is not the same as the first three.

You are given the ints n, k, and L. Fox Ciel wants to build a garden with exactly n pavilions. The garden should contain exactly k distinct simple cycles. Additionally, each of those simple cycles must have length exactly L.

Return "Possible" (quotes for clarity) if such a garden exists, or "Impossible" if there is no such garden.
