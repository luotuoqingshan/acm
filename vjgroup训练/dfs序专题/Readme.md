这个题一看是子树上面的操作，就感觉和dfs序有联系，但麻烦的是还分奇偶操作不同，但仔细观察之后发现任意一个点到它某个父亲的距离的奇偶性是到根的奇偶性和父亲到根的奇偶性的xor。于是相对性就转化为了绝对性，于是只要维护两棵线段树即可。
